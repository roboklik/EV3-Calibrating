## Use this extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.mindstorms.com/](https://makecode.mindstorms.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/roboklik/calibrating** and import

## Edit this extension ![Build status badge](https://github.com/roboklik/calibrating/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://makecode.mindstorms.com/](https://makecode.mindstorms.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/roboklik/calibrating** and click import
